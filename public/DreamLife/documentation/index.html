How to change: Background

style.css open with a text editor and change this line with your image file name.

/*for pattern*/
background:url('../images/pattern/pattern.png');
/*for fixed*/
background:url('../images/bg/.png') no-repeat 50% 50%; 
background-attachment: fixed;
background-size: cover;
